MACHINE LEARNING ENGINES

Models powering AgroSense predictions.

AgroSense uses dedicated machine learning engines for crop recommendation, price forecasting, and agro-zone classification.

Random Forest Classifier

Used for crop recommendation by evaluating soil nutrients, pH, rainfall, humidity, and temperature patterns.

Random Forest Regressor

Used for commodity price prediction by learning from historical market, crop, location, and time-based data.

Logistic Regression

Used for agro-zone classification by mapping environmental and farm inputs to suitable agricultural zones.
